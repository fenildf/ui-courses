Android 中的文字大小设置

xml 中声明 <TextView
    android:text="用户登录"
    android:textSize="20dp" />
程序中指定 textview.setTextSize(18); 注意，这里默认的单位是 sp，这是一个坑。
为什么说是坑？因为 getTextSize 方法返回的数据，单位是 px ..

sp 是啥米意思？
因为 android 允许用户设置文字尺寸，能设成小，正常，大，超大等。
当设置为“正常”时，1sp = 1dp
设置为其他尺寸时，sp会变化，dp不会变化

如果想避免坑，可以用带单位的API textview.setTextSize(int unit, int size);
// TypedValue.COMPLEX_UNIT_PX : Pixels
// TypedValue.COMPLEX_UNIT_SP : Scaled Pixels
// TypedValue.COMPLEX_UNIT_DIP : Device Independent Pixels
 来设置文字大小
