在网页中通过CSS声明字体

通过 font-family 声明来指定某个选择器下的字体。
body { font-family: 'arial'; } 指定中文字体。
body { font-family: '微软雅黑'; } 同时指定多个字体时，系统会优先尝试以在前面的字体显示文字；
如果没有指定的字体，或者字体中不包括要显示的字符，就会以声明中后面的字体来显示。 body { font-family: 宋体, 'sans serif', tahoma, verdana, helvetica; } 进一步的信息可以参考：
